# Disparador d3 🚀

Este é o sistema de disparos local integrado ao Supabase.

## 📋 Pré-requisitos

- Node.js instalado
- Conta no Supabase (configurado no `.env`)

## 🛠️ Instalação e Execução

1.  Instale as dependências:
    ```bash
    npm install
    ```

2.  Inicie o servidor local:
    ```bash
    npm run dev
    ```

3.  Acesse: `http://localhost:3000`

## 📂 Funcionalidades

- **Dashboard Premium**: Visualização moderna e responsiva.
- **Importação de Leads**: Importe facilmente listas pré-formatadas.
- **Vínculo com Empresa**: Cada lead é registrado com a empresa responsável pelo disparo.
- **Integração Supabase**: Dados persistentes e seguros.

## 📝 Importando a Lista Fornecida

O sistema já possui um botão **"Importar Lista"** no Dashboard. 
Basta informar o nome da empresa e clicar em confirmar para carregar os contatos fornecidos.
